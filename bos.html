Explicit Business Objects

User Form
// for creating and editing users
// contains fields for all fields that the user will enter

User Query
// record for a single user for view page
// includes all fields that the user will be shown on the page

All User Query
// to be used when listing all users on home page
// just needs user's name and ID

Id Query
// field with id and possibly nothing else
// used when querying or submitting based on id

SearchForm
// contains string field for the search string
// additional fields could allow advanced search functionality

SearchQuery
// record of search results returned back toward user
// an array of these would be returned to action
